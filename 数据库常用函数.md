##### COUNT汇总行数

有几种不同的语法：

* count（*）：返回表中的记录数
* count（distinct 列名）：返回指定列的不同值的数目
* count（列名）：返回指定列的值的数目（NULL不计入）

```mysql
select count(*) from 表名;
select count(distinct 列名) from 表名;
select count(列名) from 表名;
```

##### GROUP BY分组

用于结合合计函数，根据一个或多个列对结果集进行分组

```mysql
select 列名A,统计函数(列名B)
from 表名
where 查询条件
group by 列名A;
```

* 实例

    * 获取persons表中住在北京的总人数，根据lastname分组

    ![image-20260208232543407](C:\Users\666\AppData\Roaming\Typora\typora-user-images\image-20260208232543407.png)

不分组的话则会报错，即ORA-00937不是单组分组函数的错误

##### HAVING句尾连接

在SQL中增加HAVING子句的原因是，where关键字无法与合计函数一起使用

```mysql
select 列名A,统计函数(列名B)
from 表名
where 查询条件
group by 列名A
having 统计函数(列名B) 查询条件;
```
